JavaScript・ピアノ

画像をクリックすることで作成物をご覧いただけます。

コードは githubからご覧いただけますと幸いです。

※動作テストを行う際は、音量にご注意ください。

・作成期間

　3週間(50時間程度)


・作成目的

　新たに学んだ言語で成果物を作成する。

　web環境に慣れる。また、今までの経験を活かすため、入力デバイスなども扱ったものを作る。


・課題内容

　JavaScript・jQueryでピアノアプリを作成。

　ドレミファソラシと書いてあるボタンを用意し、ボタンをクリックすると鍵盤の色を変え、音名表示エリアに音名を1秒表示するようにする。

　自動演奏ボタンが押されたら、きらきら星かチューリップのどちらかをランダムで演奏させる。演奏中止ボタンを押すと自動演奏を止める。


・工夫点

　ボタンは、鍵盤を模した作りにした。

　WebAudioAPIを使用して、鍵盤が押された時に音が出るようにした。また、音量バーでボリュームを変えられるようにした。

　KeyboradEventを使い、指定のキーボードが押された時と離された時にイベントを発生させ、キーボードを押している間に鍵盤が動作するようにした。


・操作手順

　音量バーを調整→鍵盤をクリックして演奏

　音量バーを調整→自動演奏をクリックして自動演奏→演奏中止をクリックor演奏終了で自動演奏中止

　音量バーを調整→キーボードのz,x,c,v,b,n,m,s,d,g,h,jを押すことで演奏


・開発環境

　言語/JavaScript・jQuery・HTML・CSS

　API/Web Audio API・Keyboard Event

　エディタ/Visual Studio Code

　実行環境/Google Chrome
