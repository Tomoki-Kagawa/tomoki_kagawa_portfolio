JavaScript・ピアノ

画像をクリックすることで作成物をご覧いただけます。

コードは githubからご覧いただけますと幸いです。

※動作テストを行う際は、音量にご注意ください。

・作成期間

　3週間(50時間程度)


・課題内容

　JavaScriptでピアノアプリを作成。

　ドレミファソラシと書いてあるボタンを用意し、そのボタンをクリックすると音名表示エリアに音名を1秒表示させる。

　自動演奏ボタンを押すときらきら星かチューリップのどちらかをランダムで演奏させる。演奏中止ボタンを押すと自動演奏を止める。


・工夫点

　ボタンを鍵盤を模した作りにしました。

　WebAudioAPIを使用して、鍵盤が押された時に音が出る様にしました。また、音量バーでボリュームを変えられるようにしました。

　KeyboradEventを使い、キーボードで鍵盤操作できるようにしました。


・操作手順

　音量バーを調整→鍵盤をクリックして演奏

　音量バーを調整→自動演奏をクリックして自動演奏→演奏中止をクリックor演奏終了で自動演奏中止

　音量バーを調整→z,x,c,v,b,n,m,s,d,g,h,jをキーボード入力


・開発環境

　言語/JavaScript・jQuery・HTML・CSS

　API/Web Audio API・Keyboard Event

　エディタ/Visual Studio Code

　実行環境/Google Chrome
