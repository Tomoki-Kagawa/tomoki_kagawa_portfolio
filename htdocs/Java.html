Java・イライラ棒

画像をクリックすることで作成物の動作テスト動画をご覧いただけます。

コードは githubからご覧いただけますと幸いです。

※動作テストをご視聴の際は、画面の光にご注意ください。

・作成期間

　2.5週間(50時間程度)


・作成目的

　今まで学んだ言語を使って成果物をする。


・課題内容

　スマートフォン内臓の加速度センサを扱ったイライラ棒のアプリケーションを作成する。

　Androidスマートフォンを傾けることによりボールの操作を行う。


・工夫点

　UIは少なめにし、メニュー画面からの遷移方法がシンプルになるようにした。

　ステージは乱数でランダム生成をした。乱数生成時にうまくいかない場合、処理が終わらなくなり処理落ちしてしまう為、乱数生成回数とステージ生成回数に制限をかけてアプリケーションが落ちないようにした。

　実際のイライラ棒で失敗した時に電撃が走るのをスマートフォンのバイブレーション機能で再現した。


・操作手順

　スタートボタンを押す→難易度を選ぶ→スタートボタンを押し画面を傾けて操作する→ゴールまたは失敗後に次のページへ→スタートへボタンを押してスタート画面へ


・開発環境

　言語/Java・XML

　IDE/Android Studio

　使用センサ/加速度センサ・タッチパネル・バイブレーション機能

　実行環境/エミュレータ・SO-52C(実機)
